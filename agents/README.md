# Posture Guy Mike — Agent System

This directory contains specialized AI agents for the postureguymike.com Shopify redesign project. Each agent is a `.md` file loaded as context when working on that specialty area.

## How to Use
1. Tell Claude which agent context to use (e.g., "Use the Web Developer agent to build the social proof section")
2. Or reference a specific agent file directly
3. The Project Manager agent (00) has a full overview for coordination tasks
